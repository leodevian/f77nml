- License: MIT

## Installation

Install `f77nml` using [pip](https://pip.pypa.io/en/stable/):

```console
$ pip install f77nml
```

## Invocation

Invoke `f77nml` using [uvx](https://docs.astral.sh/uv/):

```console
$ uvx f77nml
```
